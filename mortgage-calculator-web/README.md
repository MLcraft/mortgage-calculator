# BC Mortgage Payment Calculator Webpage

Developed by Michelle Liu - `https://github.com/MLcraft`

Written in React with Next.js

## Description
Frontend Web UI used for communicating with BC Mortgage Payment Calculator API backend to process requests and display calculated results to the user.

**File structure:**

- All logic is found in `pages/index.js`

## Installation

```bash
$ npm install
```

## Running the app

The app will run on port `3000` by default.
```bash
# development
$ npm run dev

# build
$ npm run build

# start
$ npm run start
```
